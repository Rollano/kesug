How to Install XFCE Desktop with X11 in Termux (No Root)

This tutorial will guide you through the process of installing the XFCE desktop environment along with X11 on Termux without needing root access.

Step 1: Update and Upgrade Termux Packages

First, update and upgrade your Termux packages to ensure everything is up to date:

pkg update && pkg upgrade

Step 2: Install Necessary Packages

Next, install the necessary packages for XFCE and X11:

pkg install xfce4 xfce4-terminal pkg install x11-repo pkg install xorg-server

The xorg-server package provides the X11 server that allows you to run graphical applications.

Step 3: Set Up VNC Server

To display the XFCE desktop environment, you'll need a VNC server. Install it with:

pkg install tigervnc

Step 4: Configure VNC Server

Now, set up the VNC server by configuring the display resolution and starting the server:

vncserver -localhost

You'll be prompted to set a password. Remember this password as you'll need it to connect to the VNC server.

Step 5: Start XFCE Desktop

Create a startup script to start the XFCE desktop environment whenever you start the VNC server:

echo "startxfce4 &" > ~/.vnc/xstartup

Make the script executable:

chmod +x ~/.vnc/xstartup

Step 6: Connect to the VNC Server

To access your XFCE desktop, you'll need a VNC viewer. Download a VNC viewer app from the Play Store, like VNC Viewer by RealVNC.

Open the VNC viewer app and connect to the following address:

localhost:1

Enter the password you set earlier to access your XFCE desktop.

Troubleshooting

  • If you encounter issues with the VNC connection, ensure that the VNC server is running and that you've correctly entered the VNC password.
  • If the XFCE desktop doesn't start, verify that the xstartup script is correctly set up and executable.

Conclusion

By following these steps, you should now have a fully functional XFCE desktop environment running on Termux with X11, all without needing root access. Enjoy your new graphical interface!
