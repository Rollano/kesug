Popular Termux Tools

  • 1. SQLMap

    An open-source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over database servers.

  • 2. RED_HAWK

    All-in-one tool for information gathering and vulnerability scanning, designed for use on websites.

  • 3. Tmux

    A terminal multiplexer that allows you to manage multiple terminal sessions from a single window, making it easier to work with multiple programs.

  • 4. PowerShell

    A cross-platform task automation solution made up of a command-line shell, a scripting language, and a configuration management framework.

  • 5. Tmux Plugin Manager (TPM)

    The most popular plugin manager for Tmux, enabling users to easily install and manage Tmux plugins.

  • 6. Metasploit Framework

    A powerful open-source framework for developing, testing, and executing exploits against remote targets.

  • 7. A-Rat

    A remote administration tool (RAT) for Android devices, allowing full control of a device from a remote location.

  • 8. CrackMapExec

    A post-exploitation tool that allows attackers to conduct various attacks against Windows machines in an Active Directory environment.

  • 9. OWASP Amass

    A tool for in-depth attack surface mapping and asset discovery, designed for security professionals.

  • 10. FFUF (Fuzz Faster U Fool)

    A fast web fuzzer written in Go, used for discovering hidden files, directories, and parameters on web servers.

  • 11. Termux Tools

    A collection of useful tools and scripts to enhance the Termux environment and workflow.

  • 12. Termux API

    Provides a collection of command-line utilities for interacting with various Android system APIs directly from Termux.

  • 13. Termux Exec

    Provides a utility for executing shell scripts in a Termux environment with support for environment variables and user settings.

  • 14. Termux Styling

    A tool to customize and style the Termux terminal interface for a more personalized experience.

  • 15. Termux Session

    Helps manage and save Termux sessions to easily resume work where you left off.

  • 16. Termux Boot

    Allows Termux to run scripts automatically at boot time, useful for automation tasks.

  • 17. Termux SMS

    Provides command-line utilities for sending and receiving SMS messages from Termux.

  • 18. Termux Micro

    A lightweight text editor designed for use within the Termux environment.
